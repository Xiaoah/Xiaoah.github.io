Hexo+GithubPage+Next 搭建博客过程

发表于 2019-02-01 | 分类于 blog搭建

​ 拖了许久的博客终于开始动工了，废话不多说了，第一篇就介绍下搭建博客的全过程。其实一直以来都很想搭个自己的博客，但因为平时比较忙、又在搭博客的多种方案中犹豫，结果拖延症发作….后来在上网上看到了这个Hexo+next主题的博客风格，感觉很清晰简洁，而且看了网上的教程感觉搭建也不难，然后这个假期终于开始动工。其实这种博客搭建过程网上教程很多，也很详细，下面主要是记录下搭建博客的过程。

Overview

​ 本博客框架主要是借助 前端Hexo博客框架+Next主题风格+GithubPage，需要提前安装好Git，注册GitHub账号，安装node.js。本系统环境为Win10。相关命令可以在git bash或者cmd下执行。

安装 Git、注册GitHub账号

​ 这个就不详细说了，可以参考网上相关的教程，可以参考廖雪峰的教程

​ 注意：因为后面要用到github远程仓库的ssh地址，因此要配置好本地Git到远程仓库的ssh，在Github的ssh setting中添加好本地的公钥

建立 GitHub Pages

​ 建立过程很简单，就是直接建立一个新的仓库，New repository，然后项目的名字设置如下：GitHub用户名.github.io，比如我的是 Xiaoah.github.io。这样它就直接被视为一个GitHub Page了。

安装 Node.js

​ 下载安装Node.js，可以官网下载 。我下的是windows平台的node-v10.15.1-x64.msi。

​ 注意：建议下载最新版（最新稳定版），之前下过旧版的，在后面Hexo的一系列操作时出现各种问题，后来重新下了最新版的，就没问题了。
